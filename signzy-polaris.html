Narration

When AI stops returning a value and starts taking a decision, the unit of governance has to move from the output to the agent itself. The harness is that unit. It translates broad agent intents into exact, schema-validated Signzy API calls — Aadhaar, PAN, corporate registry inputs formatted deterministically. It holds session state across long-horizon onboarding journeys, so the agent remembers context from document upload to final video KYC without leaking data. It isolates PII — sensitive customer data is redacted or masked before it ever reaches an external LLM provider, keeping data-residency compliance strict. And it enforces the mute switch: agent behavior is monitored against predefined compliance rules, and any unmapped action or deviation from the regulatory script halts execution and escalates to a human. The off-switch works, and it is enforced by the runtime — not by policy.

Narration

Corporate underwriting is the hardest agentic workload: a single case spans hundreds of pages across financial statements, annual reports, and collateral documents, and resolution takes days, not seconds. The naive approach — stuffing whole documents into a model's context — fails three ways: accuracy degrades on long contexts, cost scales with pages rather than questions, and no single answer is traceable to its source. Polaris inverts this: the agent decomposes the case into individual questions and queries individual passages, so every answer is cheaper, more accurate, and cites the exact page it came from. Flow-specific custom tools — Polaris supports creating them first-class — are schema-validated and governed by the harness like any platform capability. Intermediate findings persist in session state across days, contradictions between documents are flagged with evidence, and the final recommendation parks at a maker-checker gate for the underwriter.
